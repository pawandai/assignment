## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/pawandai/editor.git
   ```

2. Navigate to the project directory:

   ```bash
   cd editor
   ```

3. Install dependencies:

   ```bash
   yarn
   ```
4. Run the application locally:

   ```bash
   yarn dev
   ```
